Atom Clear Night dark UI and syntax theme 2017-02-15 11:55:45 Projects,atom,theme,dark,ui,syntax,Themes,Atom Introducing a soothing new dark theme for the Atom text editor with colors inspired by the night sky. Designed to work with Clear Night Syntax, but integrates nicely with other syntax themes. Inspired by Glacier Dark UI. Visit Clear Night UI on atom.io or view the code on github.com. \n \n
\n \n

Getting started

\n

Installation

\n apm \n
 apm install clear-night-ui
\n
\n
\n Atom \n1. Open Settings > Install \n2. Set the filter to Themes \n3. Search for `clear-night-ui` and install \n \n  \n

Activation

\n1. Open Settings > Themes \n2. Select `Clear Night` from the UI Theme drop-down menu \n \n
\n \n

Settings

\n  \n \nSwitch between 3 Layout Modes: \n \n  \n \n* Auto (default) - In Auto mode, the UI and font size will automatically change based on the window size. \n* Compact - In Compact mode the UI stays compact to leave more space for the editor. \n* Spacious - Expand the UI with Spacious mode to give the editor some breathing room. \n \n
