Set DuckDuckGo as default search in Thunderbird 2017-02-27 12:00:33 Guides,security,how to,email,duckduckgo,thunderbird
\n

For whatever reason, DuckDuckGo can still be tricky to add to some services despite being the default browser for numerous browsers. A couple years ago, Mozilla made the decision to open searches from Thunderbird in your default browser, instead of using a tab in Thunderbird itself which broke the previous method of adding DDG to the settings. However, they neglected to update their tutorials for adding more than the default search engines, and as of this writing neither has DuckDuckGo (though I will be contributing this tutorial to them).

\n

This tutorial is confirmed as working for Thunderbird 45.2 - 45.7

\n \n

Install DuckDuckGo

\n
    \n \t
  1. Download and install the Google Search for Thunderbird add-on. You can easily do this within Thunderbird by going to Tools > Add-ons > Get Add-ons
    2. \n \t
  2. Navigate to your profile library. One way to get there is Help > Troubleshooting information > Application basics > Profile Directory > Open Directory
    3. \n \t
  3. Navigate to the directory extensions/gsearch@standard8.plus.com/searchplugins
    4. \n \t
  4. Save the following file into that directory: https://duckduckgo.com/opensearch.xml \n
      \n \t
    1. Rename the file as anything you want as long as you keep the .xml suffix
      2. \n \t
    2. Make sure to leave the existing Google.xml file as is
      3. \n
    \n
    5. \n \t
  5. Restart Thunderbird
    6. \n \t
  6. Go to Edit > Preferences > General > Default Search Engine and choose DuckDuckGo! Now youâ€™re set!
    7. \n
\n

You can also put any other OpenSearch XML files into that same directory if you want to add any other search engines.
