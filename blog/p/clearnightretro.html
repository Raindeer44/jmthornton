ClearNight Retro

Jade M Thornton
May 8, 2018

After a bit of work, I'm proud to introduce ClearNight Retro, a new dark, relaxed retro theme for Atom! The set is a UI theme with matching syntax theme, but both can work separately with other dark themes like Atom's own One Dark.

The core of the theme is forked from Clear Night, and makes use of a relaxed, almost faded color scheme inspired by Gruvbox. The theme set will continue to be maintained by me and the ClearNight organization.
